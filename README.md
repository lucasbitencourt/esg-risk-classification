# ESG Risk Classification using Public Data

Projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) na Universidade Federal do ABC (UFABC).

Este repositório apresenta o desenvolvimento de um modelo de classificação de risco ESG (Environmental, Social and Governance) utilizando dados públicos de empresas e técnicas de aprendizado de máquina.

---

## Objetivo

O objetivo deste estudo é analisar a viabilidade de desenvolver modelos de classificação de risco ESG a partir de dados públicos disponíveis no mercado, utilizando técnicas de ciência de dados e aprendizado de máquina.

A pesquisa busca investigar a capacidade de diferentes variáveis financeiras, indicadores de mercado e métricas corporativas em contribuir para a predição do risco ESG das empresas.

---

## Fundamentação

O crescimento da agenda ESG tem ampliado a demanda por métricas capazes de avaliar práticas ambientais, sociais e de governança das organizações. Entretanto, diversos estudos apontam divergências entre ratings ESG fornecidos por diferentes agências (Berg, Kölbel & Rigobon, 2022).

Diante desse cenário, modelos baseados em dados públicos e técnicas de aprendizado de máquina surgem como alternativa para apoiar análises de risco e sustentabilidade corporativa.

---

## Metodologia

A metodologia adotada no estudo envolve as seguintes etapas:

1. Coleta de dados públicos de empresas listadas em mercado financeiro
2. Construção e tratamento do dataset
3. Engenharia de features
4. Treinamento de modelos de classificação
5. Avaliação de desempenho dos modelos
6. Interpretação das variáveis utilizando técnicas de Explainable AI

---

## Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Scikit-Learn
- XGBoost

- Matplotlib
- Jupyter Lab

---

## Modelos Avaliados

Durante o estudo foram avaliados diferentes algoritmos de aprendizado de máquina para classificação de risco ESG, incluindo:

- Random Forest
- Gradient Boosting
- XGBoost
- K-Nearest Neighbors (KNN)

Os modelos foram avaliados com base em métricas como:

- Acurácia
- Precisão
- Recall
- F1-score

---

## Interpretabilidade do Modelo

Para compreender a influência das variáveis no processo de decisão dos modelos, foram utilizadas técnicas de interpretabilidade baseadas em SHAP (SHapley Additive Explanations), que permitem identificar a contribuição individual de cada feature nas previsões.

Essa abordagem contribui para aumentar a transparência e interpretabilidade dos modelos de aprendizado de máquina.
